キーボード入力

以下はキーボード入力に関するシステム変数 (あらかじめ格納される値や使い型が決めれらている変数)です。 変数名はユーザーが自由に決めることができますが、 以下のようなシステム変数とは異なる名前をつける 必要があります。

名前 型 説明
key char 押されているキーの文字が格納されている。
keyPressed boolean キーボードのキーが(どれか一つでも) 押されていればtrue、そうでなければfalse

以下はkeyPressedイベントの使用例です。上記で紹介した変数と 同じ名前ですが変数とイベントは異なるものです。

イベントは呼び出されるタイミングがあらかじめ決められている関数です。 システム変数と同様に同じ名前の関数を作ることはできないので注意 してください。 

			    
void keyPressed() {
  if (key == ‘c’) {  /*キーボードのCキーが押されたとき*/
    background(255, 255, 255);
  }
}

練習問題

Aキーが押されているとき、円が左から右に進むプログラムを作成してみましょう。 以下のプログラムを写し、 コメントアウトされている部分を変えて実行してみてください。

ヒント: 変数keyPressedとkeyを上手く使いましょう。 

			    
/*変数の宣言*/
float x = 0.0;
float speed = 1.0;

/*イベントの宣言*/
voidsetup(){
  size(400, 400);
}

voiddraw(){
  background(200, 200, 200);
  if(/*この部分の条件式を考える*/){
    x += speed;
}
