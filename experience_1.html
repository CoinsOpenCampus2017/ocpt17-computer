描画

Processing というプログラミング言語では簡単な命令で図形を描くことができます。試しに円を描いてみましょう。

以下のコードを書き写してみてください。 

                            
size(400, 400);
stroke(255, 0, 0);
ellipse(200, 200, 300, 300);
引数 説明
size width, height 幅(width)と高さ(height)を指定してキャンバスを作成します。
stroke R, G, B 3行目の命令で使う線の色をRed(0~255),G(0~255),B(0~255)で指定します。
ellipse x, y, rad_y, rad_x 中心の座標(x, y)と縦の直径(y_rad)、横の直径(x_rad)を指定して楕円を描画します。

サンプルコード

background(R, G, B)はキャンバスを指定した色で塗りつぶします。 

			    
size(400, 400);
background(0, 255, 255);

line(start_x, start_y, end_x, end_y)は指定した点から点へ線を引きます。 

			    
size(400, 400);
line(0, 0, 400, 400);

rect(x, y, width, height)は左上の角の座標と幅、高さを指定して長方形を描画します。 

			    
size(400, 400);
rect(100,100,200, 200);
