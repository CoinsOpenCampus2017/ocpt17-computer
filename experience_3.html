条件分岐

プログラミングをしていると、 「条件Aが成り立てばXを実行し、成り立たなければYを実行する」というような、 条件によって動作を変えたい場面が登場します。

この条件分岐を行うためにifというキーワードを用いてプログラムを書きます。 先ほど説明したような条件分岐を行うためには、以下のように書く必要があります。 

			    
if (条件A) {
  // X の処理
} else {
  // Y の処理
}

Aの部分は条件式と呼ばれる形で描く必要があります。 Aに入る条件式の基本的な形を下表にまとめました。 SやTの部分には変数や計算式や数値を書くことが出来ます。

条件式 意味
S == T SとTが等しい
S != T SとTが異なる
S <= T SはT以下である
S >= T SはT以上である
S < T SはTより小さい
S > T SはTより大きい

使用例

ステップ２で使ったmouseX. mouseYを使って簡単な条件分岐を行ってみます。 mouseXが200より小さいとき、 すなわちキャンバスの左にマウスポインタがある場合に 処理Xを実行するようなプログラムを考えると、次のように書くことが出来ます。 

			    
if (mouse < 200) {
  stroke(255, 0, 0);  // 線の色を赤にする
}

以上のように、Yの処理が必要ない場合はelse{}の部分を省略することが出来ます。 ゆえに例えば、mouseXが200未満のときのみ線の色を赤にするという処理Xの他に、 mouseXが200以上の時は青にしたい処理Yを加えるときなどは、 次のようにelse{}を挿入すればよいことになります。 

			    
if (mouse < 200) {  /* マウスポインタがキャンバス中央より左に位置しているとき */
  stroke(255, 0, 0);
} else {                 /* 上記以外(キャンバス中央より右)のとき */
  stroke(0, 0, 255);
}

練習問題

mouseXの他にmouseYを用いてキャンバスを4分割する条件式を代入し、 それぞれ出力される線の色を変えてみましょう。 ちなみに「AかつB」や「AまたはB」という条件式やさらに条件を増やしたいときは 以下のように書くことができます。 

			    
if (A && B) {           /*「&&」はプログラムにおいて「かつ」を意味します*/
  // X の処理
} else if (C || D) {    /*「||」はプログラムにおいて「または」を意味します*/
  // Y の処理
} else {                /*「AかつB」、「CまたはD」いずれにも当てはまらない場合*/
  // Zの処理
}
