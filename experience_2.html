イベント

イベントはあらかじめ呼ばれるタイミングが決められている関数です。 それらと同じ名前の関数を知らずに作ってしまうと、 意図しないタイミングで関数が呼ばれてしまうため注意が必要です。

以下はよく使うイベントです。これらはいずれも引数は無く、 返り値もありません。

名前 呼ばれるタイミング
setup プログラムが実行された時
draw 画面を描画する時
mouseDragged マウスがドラッグされた時

以下は上記のイベントの使用例です。

ドラッグしている間キャンバスが黒色になるプログラムです。 コメントアウト(//〇〇......、/*○○...*/)されているもの プログラム処理として読み込まれないため、写す必要はありません。 

			    
void setup(){
  size(400,400);
}

void draw(){
  background(255, 255, 255); // 常に背景を白にする
}

void mouseDragged(){
  background(0, 0, 0); // マウスを動かしている間のみ黒にする
}

ドラッグしている間マウスポインタの座標に合わせて線を引くというものです。 実際に入力して動かしてみてください。 

			    
void setup(){
  size(400,400);
  background(255, 255, 255);
}

void draw(){
  // このプログラムでは何もしません
}

void mouseDragged(){
  line(pmouseX, pmouseY, mouseX, mouseY);
}

2つ目の例のmouseDraggedの中で使用されているpmouseX, pmouseY, mouseX, mouseY という変数はシステム変数と呼ばれ、 あらかじめ格納される値が決められている変数です。これらは マウスの座標が勝手に格納される変数くらいに思っておいてください。
