プログラミング言語

プログラムを書くために作成された人工的な言語です。 今回使用するprocessingもその一つです。 Java, C, Pythonなど世の中には無数にプログラミング言語が存在します。

変数

変数とは値をしまっておける箱です。

型

変数が箱だとすると、型は箱の大きさと形です。 以下に代表的な型をいくつかまとめておきます。

型 説明
boolean 真偽値(true, false)。条件分岐などで利用されます。
char 文字。文字列ではありません。
color 0から255までの整数値を3つ。それぞれR,G,Bを表す。
double 64bitの浮動小数点数
float 32bitの浮動小数点数
int -2^31から2^31-1までの整数値
long -2^63から2^63-1までの整数値

今日の体験でもbooleanやintは使うことになると思います。 型ごとに使用されるデータ領域の大きさも異なります。 intとlongは同じ整数ですが使用するbit数が違うので表現できる 区間の大きさも違います。

関数

プログラミングでいう関数は(言語ごとに多少異なりますが) 一般に高校数学でいう関数とは違います。 あまり正確な表現ではありませんが、 「何度も実行す命令や、長い命令に名前をつけてまとめたもの」 程度に認識しておいてください。

引数と返り値というものがあり、引数を与えて呼び出すと 決められた命令を行います。 それが計算なら返り値は計算の答えになります。 何も返さない関数も定義できます。

引数と返り値は型が決められており、それに違反したするような形では 関数を利用することはできません。 言語によっては型の宣言が明示的でなく、このことが当てはまっていないよう に見えるものもありますが、今回使用する言語は明示的な宣言なので 気にする必要はありません。

以下に非常に簡単な関数の例を示します。整数同士の足し算をします。 

			    
int add(int a, int b) {
  return a + b;
}

returnというキーワードは非常に重要です。 このキーワードの後に来る値(ここではa+b)を返り値として返す ことを意味しており、この行が実行されるとそこで関数は終了します。

次の例は引数を取らず何も返さない関数の例です。 

			    
void sloth() {
}

このように何もしないような関数も定義できます。 引数を取らないので()の中身は空で、返り値の型はvoidとなります。
